## 二、布局结构说明
本次界面采用**三层嵌套布局**，结构清晰、层次分明：
1.  **最外层**：`Column` 作为根布局，设置浅绿色背景 `#E6F7F0`，实现整体居中布局
2.  **中间层**：分为上下两个独立模块
    - `CardTop`：垂直排列圆形头像、姓名、职位，实现个人信息展示区
    - `CardBottom`：垂直排列三行联系方式，实现联系信息展示区
3.  **最内层**：`Row` 封装 `ContactRow` 组件，实现「图标+文字」的单行联系方式布局

使用的核心组件：
- 布局组件：`Column` / `Row` / `Box` / `Spacer`
- 展示组件：`Image`（自定义头像）、`Text`、`Icon`（Material 图标）
- 样式修饰：`Modifier` 实现背景色、内边距、尺寸、圆形裁剪等样式定制

## 三、遇到的问题与解决过程
### 1. 头像显示畸形、拉伸问题
- **问题**：直接使用 `Image` 组件时，自定义头像出现拉伸、变形，圆形裁剪效果异常
- **解决**：使用 `Box` 容器包裹 `Image`，添加 `Modifier.clip(CircleShape)` 实现圆形裁剪，配合 `ContentScale.Crop` 保证图片按比例填充，完美适配圆形头像

### 2. 背景色与文字颜色不匹配
- **问题**：初始深色背景下，文字对比度低、可读性差
- **解决**：将背景改为柔和浅绿色 `#E6F7F0`，文字颜色调整为深灰/深绿，提升视觉舒适度与可读性

### 3. 代码导入报错、语法错误
- **问题**：复制代码后出现 `Unresolved reference 'Preview'` 等导入错误
- **解决**：补全 `@Preview` 注解的导入包，修正参数顺序与语法，确保代码可直接运行

### 4. 模拟器截图遮挡问题
- **问题**：Android Studio 侧边栏遮挡模拟器界面，无法获取完整无遮挡截图
- **解决**：分离模拟器窗口为独立窗口，最大化后截图，或使用布局编辑器预览截图，获取干净完整的界面

## 四、样式优化说明

3.  **文字样式优化**：姓名使用粗体大字号，职位使用绿色强调，联系方式使用清晰可读的常规字号
4.  **图标样式优化**：统一使用 Material 图标，设置绿色 tint 与整体配色统一
5.  **布局优化**：整体居中对齐，使用 `Spacer` 控制间距，界面更美观、更专业
