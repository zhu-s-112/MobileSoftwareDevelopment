# Lab5 实验报告
## 一、应用内容说明
本次实验开发了一个艺术作品浏览应用，内置3幅梵高的经典作品，用户可以通过Next/Previous按钮循环切换浏览，图片与作品名称、艺术家、创作年份同步更新。

## 二、界面结构说明
应用整体采用垂直 `Column` 布局，分为3个区块：
1.  **艺术作品墙**：用 `Surface` + `border` 实现画框效果，内部嵌套 `Image` 组件展示作品。
2.  **作品说明区**：用 `Column` 布局，包含作品标题、艺术家和创作年份。
3.  **控制按钮区**：用 `Row` 水平布局，放置Previous和Next两个按钮，实现切换功能。

## 三、状态管理实现
使用 `remember` + `mutableStateOf` 定义 `index` 状态变量，记录当前展示作品的索引。点击按钮修改 `index` 后，Compose会自动重组界面，实现图片和文字的动态刷新。

## 四、按钮切换逻辑说明
使用 `when` 表达式实现循环切换逻辑：
- **Next按钮**：如果当前是最后一幅作品，点击后回到第一幅；否则索引+1。
- **Previous按钮**：如果当前是第一幅作品，点击后跳转到最后一幅；否则索引-1。

## 五、遇到的问题与解决过程
1.  **问题1：图片资源无法识别，报`Unresolved reference: drawable`**
    解决：导入图片后，手动导入项目自动生成的 `R` 类，并同步Gradle项目。
2.  **问题2：应用运行崩溃，提示`App keeps stopping`**
    解决：修正代码结构，确保所有 `@Composable` 函数定义完整，没有语法错误。
3.  **问题3：作品切换时出现越界错误**
    解决：完善 `when` 表达式的边界处理，实现首尾循环逻辑，避免索引超出列表范围。
