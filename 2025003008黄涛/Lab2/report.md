## 三、界面布局与实现思路
整个应用采用单一页面结构，使用 Jetpack Compose 纯代码实现，无 XML 布局文件。

### 1. 整体布局结构
 页面根布局使用 `Surface` 组件，`fillMaxSize` 占满全屏。
- 背景颜色设置为深蓝色：`Color(0xFF073042)`。
- 整体内容水平居中，垂直方向使用 `Arrangement.SpaceBetween` 实现上下分布。
- 上半部分顶部留白 `120.dp`，下半部分底部留白 `40.dp`，确保视觉平衡。

### 2. 上半部分（CardTop）

- 使用 `Column` 垂直布局，内部设置 `horizontalAlignment = Alignment.CenterHorizontally`。
- 顶部展示 `Image` 头像，大小设置为 `120.dp`。
- 姓名使用 `Text` 组件，字体大小 `32.sp`，字重 `FontWeight.Bold`，颜色白色。
- 职位使用 `Text` 组件，字体大小 `18.sp`，颜色设置为绿色 `Color(0xFF3DDC84)`。
- 使用 `Spacer` 控制图片与文字、文字与文字之间的间距（`height = 24.dp` 和 `8.dp`）。

### 3. 下半部分（CardBottom）

- 使用 `Column` 垂直布局，展示三行联系方式。
- 每行之间使用 `Divider` 分割线分隔，增强视觉层次感。
- 每一行封装为 `ContactRow` 可组合函数，实现代码复用，提高可维护性。
- 每行内部使用 `Row` 横向布局：左侧 `Icon`，右侧文字信息。
- 图标统一使用绿色 tint `Color(0xFF3DDC84)`，文字为白色，保持风格统一。
- 行间距通过 `padding(vertical = 12.dp)` 控制，左右内边距为 `40.dp`。
