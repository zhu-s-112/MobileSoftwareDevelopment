# Lab2 实验报告：使用 Jetpack Compose 构建名片应用

## 一、实验目的
1. 学习并掌握 Jetpack Compose 声明式 UI 的基本开发方式。
2. 熟练使用 Column、Row 基础布局组件进行界面结构搭建。
3. 掌握 Image、Icon、Text 等基础组件的使用与样式配置。
4. 学会使用 Modifier 修饰符控制大小、边距、对齐、填充等界面属性。
5. 能够完成一个完整的名片应用，并正确处理资源加载、布局对齐、闪退等常见问题。

3. 联系电话：13187640030
4. 电子邮箱：771187485@qq.com

6. 自定义头像图片（logo.png）

## 三、界面布局与实现思路
整个应用采用单一页面结构，使用 Jetpack Compose 纯代码实现，无 XML 布局文件。

### 1. 整体布局结构
- 页面根布局使用 Column 组件，`fillMaxSize` 占满全屏。
- 背景颜色设置为浅薄荷绿：`Color(0xFFE2F4E6)`。
- 整体内容水平居中、垂直合理分布，界面简洁整齐。

### 2. 上半部分（CardTop）
- 使用 Column 垂直布局，内部水平居中。
- 顶部展示 Image 头像，大小设置为 120.dp，添加圆形裁剪修饰符。
- 下方依次显示姓名（大号黑色加粗字体）、职位（绿色字体）。
- 使用 Spacer 控制图片与文字、文字与文字之间的间距。

### 3. 下半部分（CardBottom）
- 使用 Column 垂直布局，展示三行联系方式。
- 每一行封装为 ContactRow 可组合函数，实现代码复用。
- 每行内部使用 Row 横向布局：左侧 Icon，右侧文字信息。
- 图标统一使用绿色 tint，文字为黑色，保持风格统一。

### 4. 用到的核心组件
- Column：垂直布局
- Row：水平布局
- Image：展示头像图片
- Icon：展示电话、分享、邮箱图标
- Text：显示文字信息
- Spacer：控制间距
- Modifier：设置大小、填充、边距、对齐、裁剪

## 四、开发过程中遇到的问题及解决方法
1. 问题一：程序安装成功，但打开立即闪退。
   原因：Image 加载 `R.drawable.logo` 时图片不存在或路径错误。
   解决方法：将正确的图片命名为 logo.png 放入 `res/drawable` 目录，同步项目后恢复正常。

2. 问题二：报错 `Package directive does not match the file location`。
   原因：代码文件的包名与文件夹路径不一致。
   解决方法：将 package 声明改为 `com.example.article`，与项目结构保持一致。

3. 问题三：`Unresolved reference 'drawable'` 找不到资源。
   原因：没有正确导入项目对应的 R 类，系统无法识别资源。
   解决方法：导入对应包的 R 文件，资源引用正常。

4. 问题四：界面内容不居中，图标文字排版混乱。
   原因：没有给 Column 和 Row 设置 `horizontalAlignment` 和 `verticalAlignment`。
   解决方法：为布局添加对齐属性，使整体界面居中、整齐美观。

5. 问题五：运行后界面不更新，显示旧内容。
   原因：Android Studio 缓存或模拟器未完全重启。
   解决方法：重启 Android Studio 与模拟器，重新运行项目即可加载最新代码。

6. 问题六：头像无法显示为圆形，出现矩形裁剪。
   原因：未给 Image 组件添加圆形裁剪修饰符。
   解决方法：为 Image 添加 `Modifier.clip(CircleShape)` 修饰符，实现圆形头像效果。
